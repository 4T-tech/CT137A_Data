- [1]例程_四梯教研编：四梯教研编写的CT137A竞赛实训平台的例程

  - Altera程序固化：Altera程序固化流程说明

  - CT137A_SCH：CT137A竞赛实训平台原理图
  - CT137A_UM：CT137A_蓝桥杯FPGA竞赛实训平台用户手册
