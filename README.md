# CT137A_Data

四梯FPGA竞赛实训平台（FPGA型号为EP4CE6E22C8N） 资料包



## 文件结构

- Example_Data1：CT137A硬件平台例程

- Hardware_Data：CT137A硬件平台相关硬件资料
  - datasheet：CT137A硬件平台相关芯片手册
  - TestFile：功能测试程序
  - Altera程序固化说明
  - 引脚绑定关系
  - 原理图
  
- USB_Driver：USB驱动CT137A开发板与PC连接时的USB端口驱动

- Image：图片

- LICENSE：许可证

- CT137A_UM.pdf：四梯FPGA竞赛实训平台用户手册

- 用户可以在 Intel/Altera 官网注册、下载开发环境，推荐下载 Quartus II 13.1 或 Quartus Prime 23.1（Lite Edition）版本。

  - Quartus环境下载地址：https://www.intel.com/content/www/us/en/software-kit/849770/intel-quartus-prime-lite-edition-design-software-version-24-1-for-windows.html
